<h1>Text Stats </h1>

Takes in text and returns various statistics about it depending on which level the user choses at the beginning of the program

<h2> Levels of depth</h2>

The projgram will ask the user what `level of depth` they would like to see their text in 

<ol>
<li>The first level will be the basic level that <strong> most</strong> users will be using </li>
</ol>
